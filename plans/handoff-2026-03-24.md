## 3. 已确认的阻塞点
本机无法下载 Gradle Wrapper 目标发行包：
- `android-app/gradle/wrapper/gradle-wrapper.properties`
- `distributionUrl=https://services.gradle.org/distributions/gradle-8.7-bin.zip`

现象：
- `JAVA_HOME` 已修正为系统 JDK 17，不再是 JDK 配置问题。
- `./gradlew assembleRelease` 仍然卡在下载 `gradle-8.7-bin.zip`。
- 访问链路会从 `services.gradle.org` 跳到 GitHub release，当前机器网络对该链路超时。

已做的缓解：
- 将 wrapper 超时从 `10000` 调整到 `120000`：
  - `android-app/gradle/wrapper/gradle-wrapper.properties`

## 4. 下一台电脑建议直接执行
在能正常联网下载 Gradle 8.7 的机器上：

```bash
cd android-app
export JAVA_HOME="/usr/lib/jvm/java-17-openjdk"
export PATH="$JAVA_HOME/bin:$PATH"
./gradlew assembleRelease
```

如果本机 JDK 17 路径不同，替换成实际路径即可。
